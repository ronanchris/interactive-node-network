## Maintenance

1. **Regular Tasks**
   ```bash
   # Check documentation health
   npm run docs:check
   
   # Update diagrams
   npm run generate-diagrams
   
   # Full documentation update
   npm run docs:update
   ```

2. **Machine Configuration**
   ```bash
   # Verify machine setup
   npm run verify-machine
   
   # Update machine configuration
   npm run setup-machine
   ```
